# Анализ лабораторной работы №3 по Java. 
## Исходный программный код лабораторной работы приведён к выбранному стандарту оформления Goodle Java Style.
### Исправленные ошибки в оформлении

#### 1. Расположение фигурных скобок
**Нарушение:** Открывающие фигурные скобки находились на новой строке после объявления класса или метода.

**Исправление:** Фигурные скобки перенесены на строку с объявлением.

#### 2. Отступы
**Нарушение:** Использовались отступы размером 4 пробела.

**Исправление:** Отступы заменены на 2 пробела.

#### 3. Пробелы вокруг операторов
**Нарушение:** Отсутствовали пробелы вокруг бинарных операторов, операторов сравнения и операторов присваивания.

**Исправление:** Добавлены пробелы с обеих сторон от операторов.

#### 4. Пробелы между именем метода и скобкой параметров
**Нарушение:** Присутствовал пробел между именем метода и открывающей скобкой для параметров.

**Исправление:** Пробел удалён.

#### 5. Длина строки
**Нарушение:** Некоторые строки кода превышали 100 символов.

**Исправление:** Длинные строки разбиты на несколько строк.

#### 6. Имена полей
**Нарушение:** Имена полей начинались с заглавной буквы.

**Исправление:** Поля переименованы в стиле camelCase с первой строчной буквы.
